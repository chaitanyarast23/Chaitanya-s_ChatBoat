## Uyghur Automatic Speech Recognition (ASR) System

### Overview

I developed an Automatic Speech Recognition (ASR) system for the Uyghur language that converts spoken audio into text using deep learning techniques. The project focused on building speech recognition capabilities for a low-resource language with limited publicly available datasets.

### Problem Statement

Most modern speech recognition systems primarily support high-resource languages such as English, Chinese, and Spanish. Uyghur has limited speech datasets and fewer AI resources available, making accurate speech recognition a challenging task.

### Technologies Used

- Python
- PyTorch
- OpenAI Whisper
- Speech Processing
- Deep Learning
- Natural Language Processing (NLP)

### Features

- Speech-to-Text Conversion
- Audio Preprocessing
- Uyghur Language Recognition
- Deep Learning-Based Speech Recognition
- Automatic Transcription Generation
- Model Evaluation and Performance Analysis

### Challenges

- Limited training data availability
- Low-resource language support
- Accent and pronunciation variations
- Audio quality inconsistencies

### My Contribution

- Prepared and processed speech datasets
- Fine-tuned and evaluated Whisper-based models
- Built the transcription pipeline
- Analyzed model performance and recognition accuracy
- Worked with PyTorch-based deep learning workflows

### Outcome

This project strengthened my understanding of speech processing, deep learning, PyTorch, large-scale speech models, and AI solutions for underrepresented languages. It also provided practical experience in applying OpenAI Whisper for real-world speech recognition tasks.
